# 📝 GitHub 仓库创建步骤

## ✅ 步骤 4：在 GitHub 上创建新仓库

请按照以下步骤操作：

### 1. 打开 GitHub 创建仓库页面

访问：**[https://github.com/new](https://github.com/new)**

### 2. 填写仓库信息

- **Repository name**（仓库名）: 
  - 建议：`ielts-novel-flow` 或 `ielts-novel-reading`
  - 使用小写字母和连字符（不要使用空格）

- **Description**（描述，可选）: 
  - 例如：`IELTS Novel Reading Platform - 32 novels`

- **Visibility**（可见性）:
  - **Public**（公开）：任何人都可以看到
  - **Private**（私有）：只有你可以看到
  - ⚠️ **注意**：如果选择 Private，Vercel 免费版可能不支持，建议选择 Public

- **重要**：**不要**勾选以下选项：
  - ❌ "Add a README file"
  - ❌ "Add .gitignore"
  - ❌ "Choose a license"
  
  （因为我们已经有代码了）

### 3. 点击 "Create repository"（创建仓库）
