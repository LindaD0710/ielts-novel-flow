## 步骤 1：配置 Git（必需）

请提供你的 GitHub 用户名和邮箱，或者运行以下命令：

```bash
# 配置用户名（替换为你的GitHub用户名）
git config user.name "你的GitHub用户名"

# 配置邮箱（替换为你的GitHub邮箱）
git config user.email "your.email@example.com"
```
