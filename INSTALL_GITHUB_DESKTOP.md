# 📥 GitHub Desktop 安装指南

## 🎯 步骤 1：下载 GitHub Desktop

### 方法 A：通过官网下载（推荐）

1. 打开浏览器
2. 访问：**https://desktop.github.com/**
3. 点击页面上的 **"Download for macOS"** 按钮（绿色大按钮）
4. 下载会自动开始

### 方法 B：如果下载链接有问题

- 也可以访问：**https://github.com/shiftkey/desktop/releases**
- 下载最新的 macOS 版本（.dmg 文件）

---

## 🎯 步骤 2：安装 GitHub Desktop

1. 下载完成后，在 Finder 的 **"下载"** 文件夹中找到 `GitHubDesktop.dmg` 文件
2. **双击** `GitHubDesktop.dmg` 文件
3. 会弹出一个窗口，显示 GitHub Desktop 图标
4. **拖拽** GitHub Desktop 图标到 **"Applications"** 文件夹（或双击打开）
5. 等待安装完成（通常几秒钟）

---

## 🎯 步骤 3：首次启动和登录

1. 打开 **"应用程序"**（Applications）文件夹
2. 找到 **"GitHub Desktop"**，双击打开
3. 如果是第一次打开，可能会提示：
   - "GitHub Desktop" 是从互联网下载的应用
   - 点击 **"打开"**
4. GitHub Desktop 会打开，显示欢迎界面
5. 选择 **"Sign in to GitHub.com"**
6. 点击 **"Continue with browser"**（使用浏览器登录，推荐）
7. 浏览器会打开 GitHub 登录页面
8. 使用你的 GitHub 账号登录（LindaD0710）
9. 登录成功后，浏览器会提示授权，点击 **"Authorize GitHub Desktop"**
10. 回到 GitHub Desktop，登录完成

---

## ✅ 完成！

安装完成后，GitHub Desktop 会显示主界面。
